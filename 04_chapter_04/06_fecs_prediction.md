# 4.6. Прогнозирование фильтрационно-емкостных свойств по цифровым изображениям

## Введение

Фильтрационно-емкостные свойства (ФЕС) пород, такие как пористость и проницаемость, определяют способность породы содержать и пропускать углеводороды, что критически важно для оценки запасов и планирования разработки месторождения. Традиционные методы определения этих свойств основаны на лабораторных измерениях на образцах керна, которые требуют времени и ресурсов.

Прогнозирование фильтрационно-емкостных свойств по цифровым изображениям керна представляет собой перспективный подход, который может значительно ускорить процесс оценки свойств и сделать его доступным для большего числа образцов. Связь между визуальными характеристиками изображений и свойствами пород может быть выявлена методами машинного обучения, особенно глубокого обучения, способными выявлять сложные нелинейные зависимости.

В данном разделе рассматриваются подходы к прогнозированию фильтрационно-емкостных свойств по цифровым изображениям керна, включая задачи регрессии, выбор архитектур нейронных сетей, интеграцию различных типов изображений и практические аспекты применения. Особое внимание уделяется ограничениям методов и способам их преодоления.

## Связь визуальных характеристик со свойствами

### Визуальные индикаторы пористости

Пористость может быть связана с визуальными характеристиками изображений керна через различные механизмы. Текстура породы, размер и форма зерен, наличие пор могут отражаться в визуальных характеристиках изображений. Однако связь может быть сложной и нелинейной, что требует методов машинного обучения для ее выявления.

На изображениях керна пористость может проявляться через различные визуальные признаки: текстуру поверхности, наличие видимых пор, структуру порового пространства. Эти признаки могут быть выявлены методами компьютерного зрения и использованы для оценки пористости.

Однако оценка пористости по изображениям поверхности может быть ограничена, так как изображения предоставляют информацию в основном о поверхности, в то время как пористость определяется объемными характеристиками. Интеграция с трехмерными данными, такими как рентгеновская томография, может улучшить точность.

### Визуальные индикаторы проницаемости

Проницаемость является более сложным свойством для оценки по изображениям, так как она зависит не только от объема порового пространства, но и от его структуры: размера пор, их формы, связности. Визуальные характеристики изображений могут предоставлять информацию о структуре порового пространства, но эта связь может быть еще более сложной, чем для пористости.

Текстура породы на изображениях может отражать структуру порового пространства, которая влияет на проницаемость. Например, хорошо отсортированные песчаники с крупными зернами могут иметь более высокую проницаемость, чем плохо отсортированные песчаники с мелкими зернами. Эти различия могут быть выявлены методами компьютерного зрения.

Наличие трещин на изображениях может быть особенно важным для проницаемости в трещиноватых коллекторах, где трещины могут определять проницаемость независимо от пористости. Методы компьютерного зрения могут автоматически обнаруживать и характеризовать трещины на изображениях керна.

## Применение глубокого обучения

### Сверточные нейронные сети для регрессии

Сверточные нейронные сети могут использоваться не только для классификации, но и для регрессии, предсказывая непрерывные значения пористости и проницаемости по изображениям керна. Архитектуры, разработанные для классификации изображений, могут быть адаптированы для регрессии путем замены выходного слоя.

Трансферное обучение может использовать модели, предобученные на больших наборах изображений, в качестве отправной точки для обучения на данных керна. Это может улучшить обобщающую способность и снизить требования к объему обучающих данных.

Специализированные архитектуры могут быть разработаны для учета особенностей изображений керна и задач регрессии. Например, архитектуры, учитывающие информацию на различных масштабах или мультимодальные данные, могут быть более эффективны для оценки свойств.

### Многоуровневая оценка

Многоуровневая оценка может использовать информацию на различных масштабах для оценки свойств. Детали текстуры на микроскопическом уровне могут предоставлять информацию о структуре порового пространства, в то время как общая структура на макроскопическом уровне может предоставлять информацию о геологической обстановке.

Пирамидальные структуры или многоуровневые агрегации могут интегрировать информацию на различных масштабах для более точной оценки свойств. Это особенно важно для свойств, которые зависят от структуры на различных масштабах.

### Интеграция различных типов изображений

Интеграция различных типов изображений керна, таких как фотографии в видимом свете, ультрафиолетовые изображения, рентгеновские томограммы, может улучшить точность оценки свойств. Каждый тип изображения предоставляет свою информацию, которая может дополнять другую.

Мультимодальные архитектуры могут обрабатывать различные типы изображений отдельно, а затем интегрировать их представления для предсказания свойств. Это позволяет использовать всю доступную информацию для более точной оценки.

Ультрафиолетовые изображения могут выявлять наличие углеводородов, что может быть связано с насыщенностью и, косвенно, с другими свойствами. Рентгеновские томограммы предоставляют трехмерную информацию о структуре порового пространства, которая может быть особенно важна для оценки проницаемости.

## Ограничения и вызовы

### Масштабные проблемы

Оценка свойств по изображениям сталкивается с масштабными проблемами. Изображения предоставляют информацию о поверхности или ограниченном объеме, в то время как свойства определяются объемными характеристиками. Связь между поверхностными и объемными характеристиками может быть не прямолинейной.

Лабораторные измерения выполняются на небольших образцах, которые могут не полностью представлять свойства, видимые на изображениях. Это создает проблему масштаба при связывании изображений с лабораторными измерениями для обучения моделей.

Трехмерные данные, такие как рентгеновская томография, могут предоставлять объемную информацию, но требуют более сложной обработки и анализа. Интеграция двухмерных и трехмерных данных может улучшить точность, но создает дополнительные вызовы.

### Точность оценок

Оценка свойств по изображениям может быть менее точной, чем прямые лабораторные измерения, так как изображения предоставляют косвенную информацию о свойствах. Однако для многих практических задач приблизительные оценки могут быть достаточными, особенно когда прямые измерения недоступны или требуют много времени.

Оценка неопределенности предсказаний важна для понимания надежности результатов. Байесовские методы или ансамбли могут предоставлять оценки неопределенности, которые помогают принимать решения о необходимости дополнительных измерений или проверки результатов.

## Практические аспекты применения

### Создание обучающих выборок

Создание обучающих выборок для прогнозирования свойств по изображениям требует связывания изображений керна с лабораторными измерениями свойств. Качество этой привязки критически важно: ошибки могут привести к неправильному обучению моделей.

Размер обучающей выборки должен быть достаточным для обучения эффективной модели. Для регрессионных задач обычно требуется больше данных, чем для классификации, так как необходимо выявить непрерывные зависимости, а не дискретные границы между классами.

Балансировка данных по диапазонам значений свойств может быть важна для обеспечения того, что модель хорошо работает для всех диапазонов значений, а не только для наиболее частых значений.

### Валидация моделей

Валидация моделей на независимых данных критически важна для оценки их обобщающей способности. Разделение данных по образцам или скважинам более реалистично, чем случайное разделение, так как учитывает различия в геологических условиях.

Сравнение предсказаний моделей с лабораторными измерениями может показать точность моделей и выявить области для улучшения. Метрики, такие как средняя абсолютная ошибка, средняя квадратичная ошибка, коэффициент детерминации, могут использоваться для оценки качества регрессии.

Визуализация предсказаний и их сравнение с измеренными значениями может помочь понять, как модель работает и где она может иметь ограничения. Это может помочь в улучшении моделей или в определении областей, где требуется дополнительная проверка.

## Интеграция с другими методами

### Комбинация с каротажными данными

Прогнозирование свойств по изображениям керна может быть интегрировано с прогнозированием по каротажным данным для улучшения точности. Модели могут использовать информацию из обоих источников для более точной оценки свойств.

Многоуровневая интеграция может использовать детальную информацию из изображений керна для локальных предсказаний, в то время как каротажные данные предоставляют контекст для улучшения обобщающей способности. Это может создать более точные и надежные модели.

Однако интеграция создает вызовы, связанные с привязкой данных и масштабными различиями. Требуются специальные методы для правильной интеграции различных типов данных.

### Использование в качестве дополнительных признаков

Предсказанные свойства по изображениям могут использоваться как дополнительные признаки для моделей, предсказывающих свойства по каротажу. Это может улучшить точность, используя информацию из изображений для дополнения каротажных измерений.

Однако это может накапливать ошибки: ошибки в прогнозировании свойств по изображениям будут влиять на прогнозирование по каротажу. Прямое использование изображений как признаков может быть более эффективно, если это возможно.

## Заключение

Прогнозирование фильтрационно-емкостных свойств по цифровым изображениям представляет собой перспективный подход для ускорения процесса оценки свойств пород. Методы глубокого обучения могут выявлять сложные зависимости между визуальными характеристиками изображений и свойствами пород, что позволяет оценивать свойства для образцов без прямых лабораторных измерений.

Однако оценка свойств по изображениям имеет ограничения, связанные с масштабными проблемами и косвенной природой информации. Интеграция с другими типами данных, такими как каротаж или трехмерная томография, может улучшить точность и создать более надежные модели.

Это создает комплексное представление о том, как методы искусственного интеллекта могут быть применены к анализу керна для улучшения интерпретации геологических данных.
